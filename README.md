# Quick Start

## Maven Dependancy

```xml

<dependency>
    <groupId>com.github.makewheels</groupId>
    <artifactId>HttpUtils</artifactId>
    <version>1.0.0</version>
</dependency>
```

## How to use

```java
HttpUtil.get("http://url.com");
```
